Improve overall UI responsiveness and make the app feel faster and more alive, like a top-tier music platform.

Goal:
Reduce micro-delays in clicks, playback controls, and page interactions.
The app should feel snappy, responsive, and premium.

Required improvements:

1. Immediate UI feedback
- All major buttons must react instantly on click
- Apply immediate visual feedback before waiting for async/database responses
- This includes:
  - play / pause
  - Play All
  - Add Favorite
  - comment submit
  - delete confirm
  - profile save actions

2. Use optimistic UI where appropriate
- Update local UI state immediately
- Sync with backend in the background
- If request fails, rollback state gracefully

3. Reduce unnecessary React re-renders
- Audit track lists, sidebar, player, and Library/Home sections
- Prevent full-page or full-list re-renders when only one item changes
- Use:
  - React.memo
  - useMemo
  - useCallback
  - stable keys
  - smaller isolated components

4. Improve playback responsiveness
- Play/pause actions should visually toggle immediately
- Do not delay icon changes waiting for async updates
- Keep one clear global player state source:
  - currentTrack
  - isPlaying
  - queue / section context

5. Optimize lists and sections
- Avoid re-fetching whole sections after every small action
- Update local state instead when possible
- Only refresh the minimum necessary data

6. Improve image performance
- Lazy load covers and avatars
- Use appropriately sized images
- Avoid loading unnecessarily large assets in cards/lists

7. Improve perceived performance with polished transitions
- Use fast, lightweight transitions
- Buttons: ~120–180ms
- Cards/hover: ~180–220ms
- Avoid heavy or laggy animations

8. Avoid blocking UI on async operations
- Do not freeze buttons or interaction unnecessarily
- Use subtle loading states only where truly needed

9. Keep the app feeling premium
- Interactions should feel immediate
- No “dead click” feeling
- No visible lag between click and UI reaction

Success criteria:
- clicks feel immediate
- play/pause responds instantly
- Play All feels responsive
- favorites/comments/profile actions feel faster
- overall app feels more alive and polished
