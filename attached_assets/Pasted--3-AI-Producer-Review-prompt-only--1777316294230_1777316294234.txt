# 🔧 CHANGE 2 — SIDECHAIN MUST BE EXECUTABLE

Добавить:

=== SIDECHAIN EXECUTION RULE (STRICT) ===

If the word "sidechain" appears anywhere:

→ MUST generate FULL DAW INSTRUCTION BLOCK

Required structure:

SIDECHAIN SETUP:

1. Insert Compressor on Bass channel (Insert Slot 1)
2. Open Compressor window
3. Enable Sidechain button
4. Go to Kick channel
5. Open Sends section
6. Add send → "Compressor (Bass) – Sidechain Input"
7. Set Send Level to 0.0 dB
8. Return to Compressor
9. Confirm gain reduction reacts to kick

SET:

* Threshold: 2–4 dB GR
* Attack: 5–15 ms
* Release: 80–150 ms
* Ratio: 3:1 – 5:1

METER CHECK:

* Gain Reduction meter shows 2–4 dB dips

STOP WHEN:

* bass ducks on kick but remains audible

WARNING:

* if bass disappears → reduce threshold

RULE:

* NEVER mention sidechain without this block
