========================================
If sidechain appears ANYWHERE → include FULL setup:

SIDECHAIN SETUP:

1. Insert Compressor on Bass channel (Insert Slot 1)
2. Open Compressor window
3. Enable "Sidechain" button (top right)
4. Go to Kick channel
5. Open Sends section
6. Add send → select "Compressor (Bass) – Sidechain"
7. Set Send Level to 0.0 dB
8. Return to Bass Compressor
9. Play track and confirm Gain Reduction meter moves on kick hits

SET:
- Threshold: adjust for 2–4 dB gain reduction
- Attack: 5–15 ms
- Release: 80–150 ms
- Ratio: 3:1 – 5:1

METER CHECK:
- Gain Reduction meter shows 2–4 dB dips on each kick

STOP WHEN:
- bass ducks on each kick without disappearing

WARNING:
- if bass disappears → reduce threshold by 1–2 dB
